**Safe Mode** (when `AGENTS.md` exists in target vault):
- Never write to `Mnemosyne/`, `.obsidian/`, `_archive/`

- Index DB lives **outside** vault: `~/.hermes/entropicmem/index.db`
- Default: do not index `_archive/` (opt-in via `--include-archive`)

## 3. Bootstrap Command

After `skill_manage create` installs the skill, run:

```bash
python3 ~/.hermes/skills/entropicmem/scripts/entropicmem.py init \
  --vault "$ENTROPICMEM_VAULT_PATH" \
  --index-db "$ENTROPICMEM_INDEX_DB"
```

Flags:
- `--vault PATH` — explicit vault root (overrides env)
- `--index-db PATH` — SQLite FTS DB path (default: `~/.hermes/entropicmem/index.db`)
- `--force` — overwrite seed files (`AGENTS.md`, `SCHEMA.md`, `index.md`, `log.md`, `templates/`)
- `--dry-run` — print actions without writing

## 5. Write Guards (Agent Enforces)

The following paths are **read-only** to EntropicMem:

| Path | Reason |
|------|--------|
| `Mnemosyne/` | Auto-generated by `mnemosyne_to_obsidian.py` (6h cron) — human must not edit |
| `.obsidian/` | Obsidian config — corruption risk |
| `_archive/` | Historical exports, dupes, trash — searchable, not maintained |
| `.stfolder/` | Syncthing marker |

**Implementation:** `vault.py` refuses `write_note`, `append_note`, `patch` under these prefixes. `lint`, `moc`, `graph export` skip them by default.

## 8. Graph Click-to-Open (v1: Documented, Not Automated)

`graph.html` emits `entropicmem://open/<note_id>` on node click. To enable:

**Linux (xdg-open):**
```bash
# Create ~/.local/share/applications/entropicmem.desktop
cat > ~/.local/share/applications/entropicmem.desktop << 'DESKTOP'
[Desktop Entry]
Name=EntropicMem Vault
Exec=entropicmem-open %u
Type=Application
MimeType=x-scheme-handler/entropicmem
DESKTOP

# Create ~/bin/entropicmem-open (in PATH)
cat > ~/bin/entropicmem-open << 'OPENER'
#!/bin/bash
# entropicmem-open "entropicmem://open/Infrastructure/entropicmem-architecture"
ID="${1#entropicmem://open/}"
ENTROPICMEM_VAULT_PATH="/home/ufonik/Documents/Obsidian Vault" \
  entropicmem open "$ID"
OPENER
chmod +x ~/bin/entropicmem-open
update-desktop-database ~/.local/share/applications/
```

**macOS:** `duti -s com.user.entropicmem entropicmem all` + `Info.plist` with `CFBundleURLSchemes`.

**Windows:** Registry `HKEY_CLASSES_ROOT\entropicmem` → `entropicmem-open "%1"`.

**v1 Ship Criteria:** Documented in SETUP.md; `graph.html` prints click target to console if protocol unregistered. Not a blocker.
